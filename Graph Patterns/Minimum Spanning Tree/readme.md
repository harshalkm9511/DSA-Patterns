# Minimum Spanning Tree (MST) Patterns

## 1. Classic Minimum Spanning Tree

> Build the minimum-cost graph that connects all vertices without forming cycles.

* LC 1584 - Min Cost to Connect All Points
* LC 1135 - Connecting Cities With Minimum Cost
* GFG - Minimum Spanning Tree (Prim's / Kruskal's Algorithm)


## 2. Connectivity & Union-Find (DSU)

> Maintain connected components while preventing cycles using Disjoint Set Union.

* LC 684 - Redundant Connection
* LC 1319 - Number of Operations to Make Network Connected
* LC 1579 - Remove Max Number of Edges to Keep Graph Fully Traversable


## 3. Virtual Node MST

> Introduce an extra source node to transform the problem into a standard MST.

* LC 1168 - Optimize Water Distribution in a Village
* GFG - Minimum Cost to Supply Water
* GFG - Connect Cities with Power Stations


## 4. Modified MST (Critical / Forced / Second MST)

> Modify the MST by forcing, excluding, or classifying edges.

* LC 1489 - Find Critical and Pseudo-Critical Edges in Minimum Spanning Tree
* GFG - Second Minimum Spanning Tree
* GFG - Critical Edge in Minimum Spanning Tree


## 5. Offline Queries + DSU

> Answer multiple connectivity queries efficiently by processing edges and queries in sorted order.

* LC 1697 - Checking Existence of Edge Length Limited Paths
* LC 1724 - Checking Existence of Edge Length Limited Paths II
* GFG - Offline Dynamic Connectivity
